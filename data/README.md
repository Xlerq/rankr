# Data

This directory is reserved for market data used by the engineering thesis project.

Planned data layout:
- `raw/` for original end-of-day source files from Stooq or GPW.
- `processed/` for cleaned datasets and ranking-ready outputs.

The MVP focuses on GPW/WIG20 end-of-day scoring and ranking. Do not commit sensitive data or large generated datasets without deciding repository policy first.
