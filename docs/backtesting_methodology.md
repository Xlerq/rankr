# Backtesting Methodology

This document will describe how historical ranking outputs are evaluated.

The MVP backtesting scope is limited to end-of-day GPW/WIG20 data. The goal is to assess whether ranking signals are stable, interpretable, and useful for comparative analysis.

Backtesting notes should cover:
- Historical data range.
- Rebalancing assumptions.
- Transaction cost assumptions, if used.
- Evaluation metrics.
- Limitations of the analysis.

Trading automation and brokerage integration are outside the MVP scope.
