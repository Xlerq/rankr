# Scoring Methodology

This document will describe how instruments are scored and ranked in the engineering thesis project.

For the MVP, the target is GPW/WIG20 end-of-day scoring. Candidate scoring dimensions may include trend, volatility, liquidity, recent performance, and risk-adjusted return indicators.

The scoring method should be deterministic, explainable, and suitable for thesis evaluation. Machine learning is outside the MVP scope, so ranking logic should be based on documented financial indicators and transparent formulas.

Implementation details will be added gradually after methodology decisions are finalized.
