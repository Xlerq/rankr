# API Notes

This document will track planned API boundaries for the engineering thesis application.

Initial MVP API goals:
- Expose ranked GPW/WIG20 instruments based on end-of-day data.
- Provide instrument details and historical scoring outputs.
- Exchange structured data as JSON.
- Keep CSV available for analytics and import/export workflows.

The exact routes, request formats, and response formats will be defined during backend implementation. No API code is part of the initial repository structure.
