# Development Plan

This plan keeps the engineering thesis work incremental and verifiable.

1. Define project scope, thesis topic, and methodology.
2. Document data sources and expected end-of-day fields.
3. Prepare raw and processed data directories.
4. Design deterministic scoring and ranking formulas for GPW/WIG20.
5. Add backend, database, analytics, and frontend source code gradually.
6. Validate scoring output with historical data and thesis-ready notes.
7. Prepare final documentation, diagrams, and Polish thesis chapters.

No implementation is included in this initial structure.
