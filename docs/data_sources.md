# Data Sources

The MVP will use end-of-day market data for Polish stock market instruments from Stooq and/or GPW sources.

Initial data focus:
- GPW instruments, starting with WIG20.
- Daily open, high, low, close, volume, and date fields where available.
- CSV or JSON exchange formats for local processing.

Real-time feeds are outside the MVP scope. Data source assumptions, licensing notes, field mappings, and validation rules will be documented here as the project develops.
